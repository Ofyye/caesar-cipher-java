# 🔐 Caesar Cipher em Java

Projeto simples de criptografia usando a Cifra de César, desenvolvido em Java.

---

## 🚀 Funcionalidades

* 🔒 Criptografar mensagens
* 🔓 Descriptografar mensagens
* 🔤 Suporte a letras maiúsculas e minúsculas
* 🧠 Tratamento de caracteres especiais
* ⚙️ Deslocamento personalizado

---

## 🧠 O que aprendi

* Manipulação de Strings
* Estruturas de repetição (for)
* Condições (if / else)
* Lógica de criptografia
* Tratamento de erros

---

## ▶️ Como usar

1. Execute o programa
2. Digite uma palavra ou frase
3. Escolha o valor do deslocamento
4. Escolha:

   * 1 para criptografar
   * 2 para descriptografar

---

## 💻 Tecnologias

* Java ☕

---

## 📌 Exemplo

Entrada:

```
Palavra: casa
Deslocamento: 3
Opção: 1
```

Saída:

```
fdvd
```
